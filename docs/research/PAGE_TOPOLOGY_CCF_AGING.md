# Page Topology — Smooche CCF Aging Advertorial

## URL
https://smooche.com/pages/ccf-aging

## Page Type
Magazine-style advertorial / listicle for Smooche Color Changing Foundation.
Built with Replo page builder on Shopify (Palo Alto theme 5.6.0).

## Design System
- **Fonts:** futura-pt (Adobe Typekit: cqg2bea.css), Jost (Shopify CDN fallback)
- **Colors:** #000 (black), #151515 (text), #555 (muted), #f7f7f7 (gray bg), #e5e5e5 (border), #A94E63 (rose accent), #bf0505 (sale red), #22c55e (green check)
- **Layout:** Single centered column, max-width ~700px for article content, ~900px for wider sections
- **Interaction model:** Static (no scroll-driven animations, no tab switching, no carousels with auto-play)
